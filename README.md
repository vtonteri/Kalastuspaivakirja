# **Kalastuspäiväkirja**

Ohjelman avulla käyttäjä voi luoda kalastuspäiväkirjan.

## **Huomioita Python-versioista**

Ohjelma on testattu ja toiminta varmistettu Python 3.9.7 -versiolla. Aiempien versioiden kanssa toimintaa ei voida taata.
